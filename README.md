# prework-study-guide
A study guide for course pre-work.

This project was created to allow students within the coding bootcamp program to have a useful resource to refer to while in the prework phase.

## Installation

N/A

## Usage

The sections within this website include HTML, CSS, Git, and JavaScript. Use these sections in order to prepare yourself for coding bootcamp. To see what should be covered first open the Chrome DevTools by right clicking and choosing Inspect. A consel panel should open and there you should see a list of topics that are covered and where you should begin.
